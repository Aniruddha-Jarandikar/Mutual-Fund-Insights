# 📊 Performance Metrics Implemented

### ✔ Daily Return Analysis
- Daily NAV returns calculated
- Distribution visualized using histogram

---

### ✔ CAGR Analysis
Computed

- 1-Year CAGR
- 3-Year CAGR
- 5-Year CAGR

Top performing funds were identified based on long-term growth.

---

### ✔ Sharpe Ratio

Measures risk-adjusted returns.

Higher Sharpe Ratio indicates better return generated per unit of total risk.

---

### ✔ Sortino Ratio

Measures downside-risk-adjusted returns.

Unlike Sharpe Ratio, it penalizes only negative volatility.

---

### ✔ Alpha & Beta

Calculated using Linear Regression against benchmark returns.

- Alpha → Excess return generated by fund manager
- Beta → Sensitivity towards market movement

---

### ✔ Maximum Drawdown

Worst decline from historical peak NAV.

Useful for understanding downside risk.

---

### ✔ Fund Scorecard (0–100)

Composite score generated using

- 30% CAGR
- 25% Sharpe Ratio
- 20% Alpha
- 15% Expense Ratio
- 10% Maximum Drawdown

Funds ranked based on overall quality instead of single metrics.

---

### ✔ Benchmark Comparison

Compared top mutual funds with benchmark returns over 3 years.

Visual comparison helps identify funds consistently outperforming market indices.

---

# 📈 Key Insights

- Daily returns follow a near-normal distribution centered around zero.
- Large Cap and Small Cap funds delivered the highest CAGR.
- Higher Sharpe and Sortino ratios indicate better risk-adjusted performance.
- Positive Alpha demonstrates active fund management success.
- Most top-ranked funds outperformed benchmark returns.
- Composite scorecard provides balanced ranking using growth, risk, and cost efficiency.
